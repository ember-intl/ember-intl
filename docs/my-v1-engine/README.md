## What is it?

`my-v1-engine` is an engine (routable, lazy loaded). We use it to show that engines can provide translations.


## Local development

```sh
# Run the app
pnpm start

# Lint files
pnpm lint
pnpm lint:fix

# Run tests
pnpm test
```
