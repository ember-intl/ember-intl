## What is it?

`my-v1-engine` is an engine (routable, lazy loaded). We use it in [`my-classic-app`](../my-classic-app) to show that translations can be provided by engines.


## Local development

```sh
# Run the app
pnpm start

# Lint files
pnpm lint
pnpm lint:fix

# Run tests
pnpm test
```
