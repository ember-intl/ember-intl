## What is it?

`my-v2-addon` is a v2 addon. We use it to show that v2 addons can provide translations.


## Local development

```sh
# Run the addon
pnpm start

# Lint files
pnpm lint
pnpm lint:fix

# Run tests
pnpm test
```
