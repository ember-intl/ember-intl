## What is it?

`my-v2-addon` is a v2 addon. We use it in [`my-app`](../my-app) to show that translations can be provided by v2 addons.


## Local development

```sh
# Run the addon
pnpm start

# Lint files
pnpm lint
pnpm lint:fix

# Run tests
pnpm test
```
