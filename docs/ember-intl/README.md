## What is it?

`docs-app-for-ember-intl` provides the documentation site for `ember-intl`. The app is deployed on https://ember-intl.github.io/.


## Local development

```sh
# Run the app
pnpm start

# Lint files
pnpm lint
pnpm lint:fix

# Run tests
pnpm test
```
