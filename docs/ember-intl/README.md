## What is it?

`docs-app-for-ember-intl` is an Ember app. The [deployed site](https://ember-intl.github.io/) documents the API of `ember-intl`.


## Local development

> [!NOTE]
>
> To run the commands below, some packages in this monorepo must be built first.
>
> ```sh
> # From the workspace root
> pnpm prepare
> ```

```sh
# Run the app
pnpm start

# Lint files
pnpm lint
pnpm lint:fix

# Run tests
pnpm test
```
