## What is it?

`my-v1-addon` is a v1 addon. We use it in [`my-app`](../my-app) to show that translations can be provided by v1 addons.


## Local development

```sh
# Run the app
pnpm start

# Lint files
pnpm lint
pnpm lint:fix

# Run tests
pnpm test
```
