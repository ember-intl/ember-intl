## What is it?

`test-ember-intl-node` is a Node project. We use it to unit-test the files in `blueprints` and `lib` folders in `ember-intl`.

## Local development

```sh
# Lint files
pnpm lint
pnpm lint:fix

# Run tests
pnpm test
```
